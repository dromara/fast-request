    Fast Request

    为简化调试API而生

    快速上手

    基于原生

    集成 IDEA 工具，在线离线均可使用，不再需要额外的诸如 Postman 等 API 调试工具

    效率至上

    只需要简单配置即可生成方法的 API 请求，从而节省大量时间

    功能丰富

    一键生成 URL 和参数、脚本、SearchEveryWhere 搜索、API 导出到 Postman、发送 API 请求、下载文件、存储历史请求，各种功能一应俱全

    轻量、界面友好

    轻量，仅 20M 大小，并且提供一个非常直观的界面，好操作易上手
