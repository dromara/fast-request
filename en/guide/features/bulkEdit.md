# Headers batch edit

Version required:&#x20;

&#x20;This feature allows importing header parameters directly from the browser's copy for use in the plugin.

```
1.Rows are separated by new lines
2.Keys and values are separated by :
3.Prepend // to any row you want to add but keep disabled
```
