### 功能  :100:

* [x] 一键生成url和参数
* [x] SearchEveryWhere支持
* [x] API导出到Postman
* [x] 发送请求调试API(支持修改生成的参数)
* [x] 发送请求并下载文件
* [x] 域名定制及切换
* [x] 自定义参数解析
* [x] 字符串生成策略化
* [x] 保存API
* [x] 管理API(搜索过滤、修改API名称)
* [x] 一键定位历史保存的API到方法
* [x] API转curl
* [x] 自动将API关联到模块分组
* [x] 内置Json参数编辑器
* [x] API导航树
* [x] Headers分组
